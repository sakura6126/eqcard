# EQ Card

EQ Card is an interactive emotional intelligence flashcard web app.

## Features
- Flip card (front / back)
- Deck-style navigation
- Sound effects
- Responsive layout (desktop & mobile)
- Offline usage supported

## How to use
- Click the card to flip
- Use Prev / Next to navigate
- Random button to shuffle cards
